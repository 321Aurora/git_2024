# GIT学习总结

- 首先我学习了git的概念，git是一种分布式版本控制系统，用于追踪一个团队中不同成员所做的贡献，由工作区，暂存区，本地仓库，远程仓库组成
- 其次是git的基本用法，如`git init` `git add` `git status` `git commit` `git push`等等，运用这些指令，可以完成仓库初始化，文件追踪，文件提交，以及文件推送，这一整套流程
